# Change: Phase 12 - Elegant "AI-Style" UI (Claude Inspired)

## Why
The user requested a layout "like Claude" and found the previous simple design lacking modernity. "Claude-like" implies a sophisticated, text-forward design with elegant typography (Serif headings), a warm/soft color palette (Dark Charcoal/Stone), and a focus on content readability.

## What Changes
- **Layout:** **Sidebar + Main Content** structure.
    - **Sidebar:** Settings (Winner count, keywords) & Tools.
    - **Main:** A clean canvas for the "Conversation" (Scraping -> Results).
- **Aesthetic:** "Sophisticated Intelligence".
    - **Font:** `Merriweather` (Serif) for headers + `Inter` for UI.
    - **Color:** `bg-stone-950` (Deep warm dark) + `text-stone-200`.
    - **Input:** A prominent, centered input field (like a chat prompt).
- **Interactions:** Smooth, subtle fades (Framer Motion).
- **Winner Reveal:** Elegant modal with gold/warm accents.

## Impact
- **Affected Code:** `frontend/src/**`
- **Visual Style:** Elegant, Minimalist, High-Readability.
