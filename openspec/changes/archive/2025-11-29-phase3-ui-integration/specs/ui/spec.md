## ADDED Requirements
### Requirement: Web Interface
The system SHALL provide a web-based interface for scraping and running lotteries.

#### Scenario: Scrape UI
- **WHEN** user enters a URL and clicks "Scrape"
- **THEN** the UI displays the list of fetched comments

#### Scenario: Lottery UI
- **WHEN** user clicks "Draw Winner"
- **THEN** the UI highlights or lists the selected winner(s)
